The Idea

Managing documents is an essental part of many jobs and most companies end up relying on a third party product to store files. The problem that these products have is how they make their profits. They typically charge monthly fees along with costs to adding new memebers, and the worst part, the cost of storage.

At the moment buying storage with a company like AWS will cost around $0.023 per GB, whereas many document storage companies will charge around $20 per month per person for about 50 gb of storage. Although $20 is not a huge amount of money it adds up with every employee and month.

File Man has a goal of trying to eleminate the need for these third party companies and put you the user in charge of your storage.

What We Do

File Man is not a cloud based file management system. We are a file management system with a twist. The twist is that we do not actually store any files for you. File Man provides the code necessary to have a fully functioning file management system for you or your company. This means that we are not in charge of where you put your data or the servicers in which you run your website.

Here are some of the reasons why you might benefit from switching to a File Man system:

  • The code is yours. You can make any alterationg to the program.
  • You can choose which cloud based storage service you want to use.
  • You can connect to multiple databases at the same time.
  • There are levels of permissions which restricts workers from deleting or changing files.
  • You can add custom descriptors to each project for organization.
  • Makes exporting large quantities of documents very easy.
